## Eelarve kalkulaatori idee
​
Eesmärk koostada kalkulaator, mis aitaks kasutaja rahakotisisust paremat ülevaadet saada. Kulutused grupeeritud teemade kaupa, et ülevaatlikult näha, kuhu kaob raha tegelikult.
